马尔可夫性

某一状态信息包含了相关的历史，只要当前状态可知，所有的历史信息都不再需要，当前状态就可以决定未来，则认为该状态具有马尔可夫性（Markov Property）。

马尔可夫过程

又叫马尔可夫链（Markov Chain）。它是一个无记忆的随机过程，可以用一个元组<S, P>表示，其中S是有限数量的状态集，P是状态转移概率矩阵。

马尔可夫奖励过程

马尔可夫奖励过程（Markov Reward Process）在马尔可夫过程的基础上增加了奖励R和衰减系数V：<S, P, R, V>。R是一个奖励函数。S状态下的奖励是某一时刻（t）处所在状态s下在下一个时刻（t+1）能获得的奖励期望：

Rs=E[Rt+1∣St=s]R_s = E[R_{t+1}|S_t=s] Rs​=E[Rt+1​∣St​=s]

衰减系数（Discount Factor）：γ∈[0,1]\gamma\in[0, 1]γ∈[0,1]，避免无限循环。

马尔可夫决策过程

Markov Decision Process，MDP

多了一个行为集合A，元组<S, A, P, R, V>。

Pss′a=P[St+1=s′∣St=s,At=a]Rsa=E[Rt+1∣St=s,A=a]P^a_{ss'} = P[S_{t+1}=s'|S_t=s, A_t=a]\\ R^a_s=E[R_{t+1}|S_t=s, A=a] Pss′a​=P[St+1​=s′∣St​=s,At​=a]Rsa​=E[Rt+1​∣St​=s,A=a]

下一个时刻的状态St+1S_{t+1}St+1​和当前时刻的状态StS_tSt​以及动作ata_tat​有关。

状态state

agent在每个步骤中所处于的状态集合。

行为action

agent在每个步骤中所能执行的动作集合。

转移概率transition

agent处于状态s下，执行动作a后，会转移到状态s’的概率。

奖励reward

agent处于状态s下，执行动作a后，转移到状态s’后获得的立即奖励值。

策略Policy

策略π\piπ是概率的集合或分布，其元素π(a∣s)\pi(a|s)π(a∣s)为对过程中的某一状态s采取可能的行为a的概率。

agent处于状态s下，应执行动作a的概率。

一个策略定义了个体在各个状态下的各种可能的行为方式以及其概率的大小。

回报Return

回报GtG_tGt​为在一个马尔可夫奖励链上从t时刻开始往后所有的奖励的有衰减的总和。

价值函数Value Function

价值函数给出了某一状态或某一行为的长期价值。

某一状态的价值函数为从该状态开始的马尔可夫链收获的期望。

Bellman Optimality Equation

针对V*，一个状态的最优价值等于从该状态出发采取的所有行为产生的行为价值中最大的那个行为价值：

V∗(s)=max⁡aq∗(s,a)V_*(s)=\max_aq_*(s,a) V∗​(s)=amax​q∗​(s,a)

基于值函数的策略学习方法

主要分为动态规划和蒙特卡罗。

动态规划

动态规划又分为**策略迭代（policy iteration）算法和值迭代（value iteration）**算法。

策略迭代

  1. 策略评估 policy evaluation

    计算当前策略下，每个状态的值函数。可以通过Bellman方程进行迭代计算Vπ(s)V^\pi(s)Vπ(s)。

  2. 策略改进 policy improvement

    根据值函数更新策略。

值迭代

将策略评估与策略改进合并，来直接计算出最优策略。

蒙特卡罗

Q函数。Qπ(s,a)Q^\pi(s,a)Qπ(s,a)为初始状态为s，并执行动作a后所能得到的期望总回报。

Qπ(s,a)=Er∼p(r)[G(τs0=s,a0=a)]Q^\pi(s,a)=E_{r\sim p(r)}[G(\tau_{s_0}=s,a_0=a)] Qπ(s,a)=Er∼p(r)​[G(τs0​​=s,a0​=a)]

τs0=s，a0=a\tau_{s_0}=s，a_0=aτs0​​=s，a0​=a表示轨迹τ\tauτ的起始状态和动作为s，a。

蒙特卡罗方法

Q函数通过采样进行计算。

对于一个策略π\piπ，agent从状态s，执行动作a开始，然后通过随机游走的方法探索环境，并计算其总回报。

在得到Q函数Qπ(s,a)Q^\pi(s,a)Qπ(s,a)之后，进行策略改进，在新策略下采样估计Q函数，不断重复。

ϵ\epsilonϵ-贪心法

πϵ={π(s),按概率1−ϵ随机选择A中的动作，按概率ϵ\pi^\epsilon=\begin{cases} \pi(s),按概率1-\epsilon\\ 随机选择\mathcal{A}中的动作，按概率\epsilon \end{cases} πϵ={π(s),按概率1−ϵ随机选择A中的动作，按概率ϵ​

将一个仅利用的策略转为带探索的策略，每次选择动作π(s)\pi(s)π(s)的概率为1−ϵ+1∣A∣1-\epsilon+\frac{1}{|\mathcal{A}|}1−ϵ+∣A∣1​，其它动作的概率为1A\frac{1}{\mathcal{A}}A1​。

时序差分学习方法

蒙特卡罗采样方法一般需要拿到完整的轨迹，才能对策略进行评估并更新模型，因此效率较低。

**时序差分学习（temporal-difference learning）**结合了动态规划和蒙特卡罗方法：模拟一段轨迹，每行动一步（或几步）就利用Bellman方程来评估行动前状态的值。（当每次更新动作数为最大数时，就等价于蒙特卡罗方法）。

SARSA算法

State Action Reward State Action

只需要知道当前状态s和动作a，奖励r(s,a,s’)，下一步的状态s’和动作a’，其采样和优化的策略都是πϵ\pi^\epsilonπϵ，因此是同策略。

Qπ(s,a)⟵Qπ(s,a)+α(r(s,a,s′)+rQπ(s′,a′)−Qπ(s,a))Q^\pi(s,a)\longleftarrow Q^\pi(s,a)+\alpha(r(s,a,s')+rQ^\pi(s',a')-Q^\pi(s,a)) Qπ(s,a)⟵Qπ(s,a)+α(r(s,a,s′)+rQπ(s′,a′)−Qπ(s,a))

Q学习算法

Q-learning

Q(s,a)⟵Q(s,a)+α(r+γmax⁡a′Q(s′,a′)−Q(s,a))Q(s,a)\longleftarrow Q(s,a)+\alpha(r+\gamma\max_{a'}Q(s',a')-Q(s,a)) Q(s,a)⟵Q(s,a)+α(r+γa′max​Q(s′,a′)−Q(s,a))

与SARSA不同，Q-learning不通过πϵ\pi^\epsilonπϵ来选下一步的动作a’，而是直接选最优的Q函数。更新后的Q函数是关于策略π\piπ的，而不是策略πϵ\pi^\epsilonπϵ的。
