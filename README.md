## Hexo + Github 博客

现在主流的基于 Github 的博客都是根据 Jeklly 和 Hexo 两种搭建的方式，因为之前我使用的是基于 Jeklly 的方式并且不太满意，所以现在决定搭建一个基于 Hexo 的博客。

选择搭建 Github 博客的方式是托管方便，并且 Github 社区活跃，有很多相似的内容。
